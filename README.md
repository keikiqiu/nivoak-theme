# Nivoak Theme v2.1.0

Dark luxury WordPress theme for premium bar accessories.

## Design System
- Background: #0a0807 (near black)
- Surface: #1a1511, #221c16
- Accent: #c8932f (amber gold)
- Text: #f5ede0, #a89b88, #6b5f50
- Fonts: Playfair Display (serif), Inter (sans)

## Files
- `style.css` - Main stylesheet (v2.1.0)
- `functions.php` - Theme functions with breadcrumb support
- `header.php` - Header with fixed nav, JS detection, 3s fallback
- `footer.php` - Footer with correct /category/ links
- `index.php` - Homepage with hero, categories, craft, blog, CTA
- `page.php` - Page template with breadcrumb
- `single.php` - Single post template
- `mu-plugins/nivoak-fix.php` - Must-use plugin (taxonomy + breadcrumb fix)

## Deployment
```bash
bash deploy.sh
```
